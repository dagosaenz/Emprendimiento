Como funciona?

El TIME LIGHT es un sistema para temporizar el uso de la luz en zonas comunes en las cuales se necesita iluminación de una manera pasajera y por alguna razón no se puede apagar nuevamente la luz para así economizar energía. Solo se debe accionar un interruptor para encender la luz como se hace convencionalmente, pero de acuerdo a su programación esta se apagara automáticamente. El tiempo de duración puede programarse de acuerdo a la necesidad del usuario y de la zona en la cual se instale.

Fabricación

El TIME LIGHT está fabricado a base de tres componentes electrónicos principales, un TEMPORIZADOR, un RELÉ y un INTERRUPTOR, el interruptor (cono ya se conoce) se usa únicamente para accionar el sistema, el relé o relevador distribuye la corriente a todo el sistema para que se mantenga encendida la lámpara y el temporizador de acuerdo a su programación se encarga de interrumpir y cortar por completo la corriente para producir el apagado automático. Cabe anotar que este sistema es diseñado y construido en casa y los materiales utilizados para la construcción de partes como la caja donde se encuentran los componentes electrónicos y los interruptores son básicamente reciclados.

Beneficios

Este sistema está diseñado para optimizar el uso de la luz eléctrica en cualquier ambiente que lo requiera, generando así un ahorro en los costos de este servicio. Por ser un sistema construido en casa los costos se reducen considerablemente y en valor no se compara con un producto que se encuentre en el mercado. Para su instalación solo se deben tener conocimientos básicos de electricidad o para mayor efectividad ser instalado por su técnico de confianza, algo que en realidad tomara poco tiempo.
